MaterialScope Hızlı Yardım
===========================

1. Programı nasıl açarım?
- Başlat Menüsü'nden veya masaüstü kısayolundan MaterialScope Beta'yı açın.
- Program tarayıcıda açılır. Bu normaldir.
- Uygulama bulutta değil, kendi bilgisayarınızda çalışır.

2. Şu anda güvenle test edilebilen alanlar
- DSC analizi
- TGA analizi
- Karşılaştırma Alanı
- Toplu Şablon Uygulayıcı
- Rapor / dışa aktarma
- .scopezip proje kaydetme ve yeniden açma (legacy .thermozip açma desteklenir)

3. Önizleme alanları
- DTA
- Kinetik analiz
- Pik dekonvolüsyonu

Bu önizleme alanlarını kararlı çıktı gibi değerlendirmeyin.

4. Dosya yükleme
- Önerilen dosyalar: CSV, TXT / TSV, XLSX / XLS
- Bir dosya yükledikten sonra içe aktarma uyarılarını kontrol edin.
- Veri tipi, sıcaklık kolonu veya sinyal kolonu yanlış görünüyorsa kolon eşlemeyi elle düzeltin.

5. Temel kullanım akışı
- Dosyayı yükleyin
- Gerekirse kolonları düzeltin
- DSC veya TGA sayfasına gidin
- Analizi çalıştırın
- Sonucu kaydedin
- Rapor veya dışa aktarma alın

6. Rapor ve proje alma
- Rapor Merkezi sayfasından DOCX / PDF rapor alabilirsiniz.
- CSV / XLSX dışa aktarma alabilirsiniz.
- Çalışmayı daha sonra açmak için .scopezip proje arşivi kaydedebilirsiniz.

7. Sorun bildirirken ne göndermeliyim?
- Mümkünse giriş dosyası
- Ekran görüntüsü
- İlgili .scopezip dosyası (veya legacy .thermozip)
- Destek tanılama çıktısı

Destek tanılama çıktısını Rapor Merkezi içindeki destek alanından dışa aktarabilirsiniz.

8. Kısa notlar
- Program tarayıcıda açıldığı için ilk kullanımda tarayıcı penceresi görünür.
- Şüpheli bir içe aktarma varsa sonucu yorumlamadan önce dosyayı kontrol edin.
- Eksik kalibrasyon bilgisi varsa sonucu tam doğrulanmış kabul etmeyin.
